0x0B. C - malloc, free

This project introduces automatic vs dynamic memory allocation, how to use `malloc` and `free`, and how to use Valgrind to check for memory leaks.

#Files
0-create_array.c:
Creates and initializes an array of chars

1-strdup.c:
Allocate memory and copy a string to the new space in memory

2-str_concat.c:
Concatenate two strings in a dynamically allocated space in memory

3-alloc_grid.c:
Allocate space in memory for a 2D array

4-free_grid.c:
Free the memory allocated for a 2D array in 3-alloc_grid.c

5-argstostr.c:
Concatenate all the command line arguments given to it
